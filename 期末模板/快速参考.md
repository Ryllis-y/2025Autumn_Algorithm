### 图存储（链式前向星）
```cpp
int node[maxn], edge[maxn], nxt[maxn], weight[maxn], cnt = 1;
void addEdge(int f, int t, int w){
    nxt[cnt] = node[f];
    node[f] = cnt;
    edge[cnt] = t;
    weight[cnt] = w;
    cnt++;
}
```

| 算法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|

| 树状数组 | O(n log n) | O(n) |
| ST表 | O(n log n) | O(n log n) |

| SPFA | O(km) | O(n+m) |
| Floyd | O(n³) | O(n²) |
| Kruskal | O(m log m) | O(n) |
| KMP | O(n+m) | O(m) |
| Manacher | O(n) | O(n) |
| 快速幂 | O(log b) | O(1) |
