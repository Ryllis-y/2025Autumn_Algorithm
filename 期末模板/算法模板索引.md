## 🚀 快速查找指南

### 按问题类型查找

- **区间查询/修改** → 线段树、树状数组、ST表
- **最短路问题** → Dijkstra、SPFA、Floyd
- **字符串匹配** → KMP、字符串哈希
- **回文问题** → Manacher
- **网络流** → Dinic、EK、最小费用最大流
- **组合计数** → 组合数、快速幂
- **数论** → 扩展欧几里得、素数筛

### 按复杂度选择

- **O(n log n)** → 线段树、树状数组、Dijkstra、Prim、Kruskal
- **O(n²)** → Floyd、LIS (DP)
- **O(n)** → KMP、Manacher、单调栈、单调队列
- **O(log n)** → 二分查找、快速幂、LCA

## 🔧 常见问题

### 图存储方式
- 大部分图算法使用**链式前向星**存储
- 数组：`node[]`, `edge[]`, `nxt[]`, `weight[]`
- 添加边：`addEdge(f, t, w)` 或 `addEdge(f, t)`

### 初始化
- 距离数组：`memset(dist, 0x3f, sizeof(dist))`
- 访问数组：`memset(visit, 0, sizeof(visit))`
- 图数组：`memset(node, -1, sizeof(node))` 或 `memset(node, 0, sizeof(node))`
